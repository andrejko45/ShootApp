### Spustenie aplikácie

1. Vytvorte si priečinok s ľubovoľným názvom a presuňte sa do neho.
2. Zadajte v príkazovom riadku git clone https://github.com/andrejko45/ShootApp.git
3. Presuňte sa do priečinku ShootApp.
4. Zadajte príkaz docker compose up --build pri prvom spustení.
5. Aplikáciu ukončíte pomocou docker compose down.
